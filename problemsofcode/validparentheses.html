El problema de los parentesis

Hoy vamos a solucionar un reto para escribir un algoritmo que nos permita verificar si en un string tenemos una serie de parentesis correctamente abiertos y cerrados.

Definición:

Dada una cadena de texto s que contiene unicamente los caracteres '(', ')', '{', '}', '[', ']' y determinar si la salida es valida.

Una salida es valida si:

  1. Los parentesis abiertos deberan estar cerrados por el mismo tipo de parentesis
  2. Los parentesis se deben cerrar en el orden correcto

Ejemplo 1

entrada: "()"

Salida: true

Ejemplo 2

entrada: "()[]{}"

Salida: true

Ejemplo 3

entrada: "(]"

Salida: false

Ejemplo 4

entrada: "{[]}}"

Salida: true

Limites

  • 1 <= s.length <= 10000
  • s consiste unicamente '()[]{}'
